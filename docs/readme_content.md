# Contenu à inclure dans le README

Utilisation de Claude Code pour l'implémentation de ce projet, tout en gardant la main mise dessus pour ne pas avoir de dettes techniques.
En particulier sur :
- Génération des données synthétiques dans le fichier `fixtures.py`
- Génération des modèles des données reçues et envoyées à l'API Uber Eats, définies dans les fichiers `.json` du dossier `uber_eats_api_doc`.


C'est un projet personnel à travers lequel je souhaite aussi prendre du plaisir, en ne me mettant pas de pression de deadline ou autre pour avoir le temps de faire des recherches et enrichir mes connaissances, codés moi-même en utilisant peu Claude Code ou Copilot même si ça peut prendre plus de temps et être moins propre parfois, etc.
